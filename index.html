HTML

  • Hypertext Markup Language
  • Linguagem voltada para marcação/estruturação de documentos da web
  • Responsáveis pela estrutura da página web
  • Baseada em tags XML
  • As tags são interpretadas pelo browser, e então o conteúdo é exibido para o usuário

HTML - Tags

  • A linguagem possui um conjunto de tags XML utilizadas para descrever sua estrutura
  • Toda tag possui atributo e valor
				
			<a href="http://www.google.com.br">Clique aqui</a>

Tags - Atributos

  • Uma tag pode ter 0 ou mais atributos
  • Um atributo é uma tupla de dados
  • São utilizados para definirem propriedades da tag
				
		<tag attr1="valor1" attr2="valor2" attr3="valor3">...</tag>

Tags - Valor

  • Valor: conteúdo a ser exibido pela tag
  • HTML permite fazer o aninhamento de tags
				
<tag ...> valor da tag </tag>
				
			
				
<tag ...>
	<tag ...> texto </tag>
</tag>

HTML - Estrutura Básica

				
<!DOCTYPE html PUBLIC "-//W3C//DTD HTML 4.01//EN">
<html>
    <head>
         <title>Minha primeira página HTML</title>
    </head>
    <body>
         <!– meu comentário —>
         <p> Essa é a minha primeira página web. </p>
    </body>
</html>

HTML - Estrutura Básica

  • <!DOCTYPE>: Versão do documento HTML
  • <html>: delimita a página HTML
  • <head>: Cabeçalho da página. Contém informações de configuração da página
  • <body>: Corpo da página. Contém toda informação que será exibida para o funcionário

Introdução

A linguagem HTML fornece uma série de elementos que ajudam na estruturação de uma página web

Os principais elementos estão relacionados a:
  1. Manipulação de Texto
  2. Containers
  3. Formulários
  4. Elementos Multimedia

Manipulação de Texto

Manipulação de Texto

  • Headings
  • Formatação
  • Parágrafos
  • Listas
  • etc..

Multimedia

Multimedia

Elementos que permitem inserir elementos multimedia na página.

  • Link
  • Imagens
  • iFrames
  • etc..

Multimedia - link

					
<a href="http://www.google.com" target="_blank">Click me</a>.
Principais atributos:
  • href: link da página a ser carregada
  • target: define se a página será aberta em outra janela
    • _blank: nova página
    • _self: mesma página

Multimedia - Imagens

					
<img src="http://www.uol.com.br/uploads/29111.jpg" alt="2911" />
Principais atributos:
  • src: fonte da imagem
  • alt: texto alternativo da imagem

Multimedia - iFrames

				
<iframe src="http://www.uol.com.br/"></iframe>
Principais atributos:
  • src: fonte da página a ser carregada
  • width: largura do frame
  • height: altura do frame

Formulários

Formulários

  • Mecanismo de comunicação básica entre usuário e aplicação web
  • Permite ao usuário enviar suas informações para a aplicação
  • HTML possui uma série de elementos de entrada de dados, onde o usuário pode fornecer este tipo de informação

Formulários

Estrutura básica 

				
<form>
	Username: <input type="text" name="username"> <br>
	Password: <input type="password" name=“pwd"> <br>
	<input type="submit" value="Log in">
</form>

Formulários

As entradas de dados são feitas através de três tipos de componentes:

  • <input>: texto, senhas, checkbox, radio buttons, botão submit, botão reset;
  • <select>: listbox, combobox;
  • <textarea>: texto em várias linhas;

Formulários - input

				
<form>
	Username: <input type="text" name="username">
	Password: <input type="password" name="pwd">
	Sex: <input type="radio" name="sex" value="male">Male</input>
	<input type="radio" name="sex" value="female">Female</input>

	Vehicles: <input type="checkbox" name="vehicle" value="bike">Bike</input>
	<input type="checkbox" name="vehicle" value="motocycle">Motocycle</input>
	<input type="checkbox" name="vehicle" value="car">Car</input>

	<input type="submit" value="Enviar">
</form>

Formulários - input

Principais Atributos:

  • autocomplete: auto-completar
  • checked: marca elemento como default
  • name: nome do elemento
  • type: tipo do elemento de entrada
  • value: valor do elemento
  • etc, etc..
